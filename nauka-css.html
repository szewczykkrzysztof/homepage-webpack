Nauka CSS

Sposoby stylowania w CSS

Selektory

Selektorem może być dowolny znacznik np. header, h1, footer.
W css przypisuje się styl do znacznika za pomocą takie struktury:

nazwa-taga { właściwość: wartość; }

Właściwości może być tutaj więcej.

Klasy

Jak używać klas?

W pliku z kodem html należy dodać do znacznika nazwę class="name".
W ten sposób możemy pogrupować elementy za pomocą klasy.
Nazwa powinna być zapisana za pomocą kebeb-case. Aby teraz przypisać styl do tak utworzonej klasy wywołujemy go w css jako .class-name. Kropka jest tutaj selektorem klasy.

Identyfikatory

Selektorem , który możemy wykorzystać do stylowania może być również identyfikator, którego użyliśmy na stronie. Ważne, jest przy tym, aby użyć go na każdej stronie tylko raz. Jak wywołujemy taki selektor?
Przed jego nazwą stawiamy #. Przykład:

#id { właściwość: wartość;}

Łączenie selektorów

Selektory możemy też łączyć ze sobą.
Możemy to wykorzystać, aby nadać styl tagom zagnieżdżonym np.

nav a {color: example-color}

nada linkom (a) znajdującym się w sekcji nav kolor, którego nie będą miały inne linki.

Styl zostanie nie zostanie nadany dla całej sekcji nav, lecz tylko do osatniego taga w selektorze. W tym przypadku będzie to [a] czyli linki.

Łączone selektory nie muszą występować bezpośrednio po sobie. W naszym przykładzie tag linku [a] nie musiał zostać sparowany z bezpośrednim rodzicem, czyli [nav].

Następnie nadajmy mu klasę i ostylujmy go w css
Gdy element ma już odpowiedni wygląd, dodajemy mu pseudoklasę :hover. Najlepiej jest to zrobić w scss poprzez zagnieżdżenie wewnątrz klasy rodzica. W pseudoklasie definujemy które właściwośći będą się zmieniały po najechaniu myszką na element.
Na tym etapie element po wskazaniu będzię się już zmieniał, ale będzie się to odbywać skokowo, a nie o to nam przecież chodzi.
Dodajmy więc do klasy wyjściowej właściwość: <div> class"code">transition: all 1s linear </div>
