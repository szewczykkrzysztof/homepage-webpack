Nauka Javascript

Kilka słów o Javascript

Kod javascript uruchania się raz, po załadowaniu strony.
Jest wykonywany linijka po linijce, zadanie po zadaniu

Znacznik html, który wywołuje javascript wewnątrz html to script
Polecenia opakowane w ten tag umieszczamy na samym końcu strony przed tagiem body

Ponieważ plik html z kodem js rozrósł by nam się nadmiernie, lepiej kod javascriptu wydzielić sobie do osobnego pliku.
Pliki javascript mają rozszerzenie *.js.
Zewnętrzny plik zawierający skrypty wywołujemy dodając do znacznika script wartość src="scieżka-do-pliku"

Zmienne w JS przypisujemy za pomocą komendy const = wartość
Wartrości tekstowe zawieramy w ' ', natomiast liczby przypisujemy normalnie

Literały

Chcąc wykorzystać kombinację łańcuch tekstowego i zmiennych musimy skorzystać z symbolu backtik (`)

Tworzymy w ten sposób łancuch zwany literałem:

`Mam na imię ${name}`

Zmienna jest podstawiana w miejsce ${}

Pierwsze polecenia

alert

Polecenie wywołuje wyskakujące okienko

alert('Tekst napisu');

console.log

To polecenie wyświetla treści w okienku konsoli przeglądarki

Ma postać console.log('Tekst do wyświetlenia')
