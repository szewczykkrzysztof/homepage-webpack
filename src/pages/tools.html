Visual Studio Code

Przydatne skróty

Ctrl+Shift+p - wywołuje linię komend VSC

Formatowanie kodu

W linii komend VSC wpisujemy "format document". Jest to narzędzie, które uporządkuje nam wpisany kod. (lub shift+alt+f)

Prettier

Plugin do VSC wprowadza zaawansowane formatowanie kodu

Liveserver

Plugin do VSC dodaje obsługę lokalnego serwera http. Dodatkowym bonusem jest automatyczne odświeżanie strony w przeglądarce po zapisaniu pliku z kodem

GIT

System kontroli wersji, który pozwala zapanować nad wprowadzanymi modyfikacjami do naszego kodu.
